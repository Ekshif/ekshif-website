# اكشف - Ekshif

لعبة ألغاز منطقية بالعربي تجمع العائلة والأصدقاء.

## التقنيات

- HTML/CSS/JavaScript (Vanilla)
- Supabase (قادم) - قاعدة بيانات + Auth
- Moyasar (قادم) - بوابة الدفع
- Vercel - الاستضافة

## التطوير المحلي

```bash
# افتحي index.html في المتصفح مباشرة
# أو شغلي سيرفر محلي:
python3 -m http.server 8000
# ثم افتحي http://localhost:8000
```

## النشر

النشر تلقائي عبر Vercel عند كل push للـ `main` branch.
